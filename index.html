Project description

Quick Test is a web application / platform for Online Assessments, tests, surveys, quizzes and exams. It’s best suited for the corporate sector, admissions management, online evaluations and skills assessment for the educational institutions, companies, corporations and agencies.

Quick Test is a single page web application developed by Vue.js with Laravel 8 API and Mysql DB.

Set up QuickTestApi

  • First create a database on your Mysql DB then open .env file and set DB_HOST, DB_PORT, DB_DATABASE, DB_USERNAME & DB_PASSWORD to connect with that database.

 

  • Now set MAIL_MAILER, MAIL_HOST, MAIL_PORT, MAIL_USERNAME, MAIL_PASSWORD & MAIL_ENCRYPTION to configure your email settings. For Gmail you can set smtp as MAIL_MAILER, smtp.gmail.com as MAIL_HOST, 587 as MAIL_PORT, tls as MAIL_ENCRYPTION and of course set MAIL_USERNAME & MAIL_PASSWORD as yours.

 

  • For production version of your API you should set production as APP_ENV and false as APP_DEBUG.

 

  • Now execute below commands on terminal

                    ->  composer install

                    ->  php artisan migrate:fresh --seed     

           now you will see a vendor folder be added to your API project and some tables be added to your database with some migration data. 

  • Now execute "php artisan serve" on terminal and you will see your API project running at http://127.0.0.1:8000 

Set up QuickTestClient

  • Open QuickTestClient on Visual Studio Code.On terminal write "npm install", it will add node modules to project then write "npm run serve".

  • Now open config.js file, you should see a hostname here which is set to http://localhost:8000. Actually QuickTestApi project is running here.Change this url in case of actual hosting.

  • After doing above settings your client project will be running at http://localhost:8080 

  • For production build execute "npm run build" on terminal.

In Candidates Dashboard, all above charts are available only for that logged in candidate and on top of all charts there will be a list of available assessments.

Menus

Here you find a list of all menus of this App. To add a new menu you have to fill below fields

  1. Menu Title (No duplicate Menu Title is accepted)

  2. URL (No duplicate URL is accepted)

  3. Order No. (No duplicate order no. is accepted)

  4. Icon (Only material design icon is allowed, browse this link to find more icons https://fonts.google.com/icons?selected=Material+Icons)

Roles

You will find a list of two roles (Admin & Student) initially. These two roles are restricted to modify and delete because these two roles are used for many configuration issue.

Anyway you can add new roles and can modify or delete them as well.  

Users

Initially you will find two users here for testing purpose. You can modify those and can add new users from here.

Category

Here you will find a list of question category and able to add new one.

Assessments

It is the starting point of a new assessment. You can create new Exam or quiz or survey by filling below fields,

  1. Title (Title of your assessment, No duplicate is accepted)

  2. Mark Option (Three options available i.e Equal distribution,Question wise set & No marks(survey), actually here you will define whether it is an marks based exam or an survey without marks).

  3. Multiple Attempt ( Check this only If you allow your students to take a assessment multiple times)

  4. Multiple Answer ( Check this if you want to set multiple answers for one question)

  5. Multiple Attempt ( Check this only If you allow your students to select multiple answer for one question)

  6. Marks (only visible for Question wise Mark Option)

  7. Required Time (only visible for Question wise Mark Option and must be set in minutes)

  8. Pass Marks (Available for Marks based assessments and one student have to pass for getting a certificate)

  9. Schedule Start Date ( in case of schedule a assessment)

  10. Schedule End Date ( in case of schedule a assessment)

  11. Certificate Template ( select one if you want to give certificate)

  12. Correct Answer display to students ( Check this if you allow correct answers to student result sheet)

  13. Participant Option ( All registered students or you may set a comma seperated email list like abc@gmail.com, xyz@hotmail.com,def@yahoo.com)

  14. Sent invitation to email will available if you select custom input as participant option

  15. Price (set price of each assessment, not mendatory)

Questions

List of assessment will find here and you can add questions for each assessment by filling below fields

  1. Question ( No duplicate is accepted for a particular quiz)

  2. Code Snippet ( Check this only if you want set pre-formatted questions like Programming language code or anything you want keep source format)

  3. Question Type ( MCQ & Descriptive)

  4. Complexity Lavel (Easy, Medium & Hard)

  5. Question Category(select one category from your existing category list)

  6. Question Video (Only youtube embed video is allowed) (Optional)

  7. Question Image (Optional)

  8. Option-A

  9. Option-B

  10. Option-C (Optional)

  11. Option-D (Optional)

  12. Option-E (Optional)

  13. Correct Answer

  14. Answer Explanation (Optional)

(8-12) applicable for MCQ questions only.
